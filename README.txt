Shockwave 3D World Converter 1.3.2
By Anthony Kleine

	This tool converts Shockwave 3D World Files (W3D)
	to Wavefront OBJ, Material and
	Image Files (OBJ, MTL, and TIFF.)
	
	Made with Adobe Director 11.5.9.629. Requires
	LeechProtectionRemovalHelp Xtra, DropXtra,
	ImgXtra, and awaSW3D Xtra (Beta.)
	
	Cannot currently export Bones.
	
Features
	-Models
	-Normals
	-Vertex Colors
	-Texture Coordinates
	-Materials (as MTL)
	-Ambient
	-Diffuse
	-Specular
	-Shininess
	-Textures (as TIFF)
	-Texture Layers (as layered TIFF)
	-Transparency
